# Icon Switch

The `<IconSwitch />` is a switch component that can optionally renders an icon in the switch's handle.

<Box
  sx={{
    width: '100%',
    textAlign: 'center',

      width: '100%',
      maxWidth: '300px',
    }}
    component="img"
    alt="icon switch picture"
    src={iconSwitchImage}
  />
</Box>
