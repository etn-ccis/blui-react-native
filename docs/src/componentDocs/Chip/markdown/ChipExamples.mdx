## Chip

The basic chip is a clickable element displaying a text label.

<Box
  sx={{
    width: '100%',
    textAlign: 'center',
  }}>
  <Box
    sx={{
      width: '100%',
      maxWidth: '500px',
    }}
    component="img"
    alt="Chip Compoent"
    src={ChipExample}
  />
</Box>

```
<Chip>Label</Chip>
```

## Adding an Icon / Avatar

You can add an optional icon or avatar to the chip to provide more information / context.

<Box
  sx={{
    width: '100%',
    textAlign: 'center',
  }}>
  <Box
    sx={{
      width: '100%',
      maxWidth: '500px',
    }}
    component="img"
    alt="Chip Icon Component"
    src={ChipIconExample}
  />
</Box>

```
<Chip
    icon={{ name: 'info' }}
    iconColor={'#f00'}
>
    Chip With Icon
</Chip>
<Chip
    avatar={<Avatar.Icon size={40} icon="account-circle" />}
>
    Chip With Avatar
</Chip>
```

## Selecting a Chip

Chips can be selected, which changes their styling to indicate that they are active.

<Box
  sx={{
    width: '100%',
    textAlign: 'center',
  }}>
  <Box
    sx={{
      width: '100%',
      maxWidth: '500px',
    }}
    component="img"
    alt="Chip Selected Component"
    src={ChipSelectedExample}
  />
</Box>

```
<Chip>
    Unselected Chip
</Chip>
<Chip selected>
    Selected Chip
</Chip>
```
