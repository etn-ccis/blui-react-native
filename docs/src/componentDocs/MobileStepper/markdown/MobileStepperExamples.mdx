## Mobile Stepper

The basic mobile stepper uses dots to indicate progress.

<Box
    sx={{
        width: '100%',
        textAlign: 'center',
    }}
>
    <Box
        sx={{
            width: '100%',
            maxWidth: '400px',
        }}
        component="img"
        alt="Mobile Stepper Dot Variant"
        src={MobileStepperDotVariant}
    />
</Box>

```
<MobileStepper
  activeStep={activeStep}
  steps={steps}
  leftButton={<BackButton />}
  rightButton={<NextButton />}
  variant={'dots'}
/>;
```
