# Mobile Stepper

The `<MobileStepper>` is used to display progress when completing a workflow that requires multiple steps. There are three types of progress indicators supported, `dots` (default), `text`, and `progress`.

## Dot Variant Usage

<Box
  sx={{
    width: '100%',
    textAlign: 'center',
  }}>
  <Box
    sx={{
      width: '100%',
      maxWidth: '400px',
    }}
    component="img"
    alt="Mobile Stepper Dot Variant"
    src={MobileStepperDotVariant}
  />
</Box>

```
<MobileStepper
  activeStep={activeStep}
  steps={steps}
  leftButton={<BackButton />}
  rightButton={<NextButton />}
  variant={'dots'}
/>;
```
