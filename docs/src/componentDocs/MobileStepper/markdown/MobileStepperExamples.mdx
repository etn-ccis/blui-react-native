## Using Text / Progress Bar

If you have more than a few steps or limited space in your container, you can use a text label or progress bar to indicate the current step.


<MobileStepper
  activeStep={activeStep}
  steps={steps}
  leftButton={<BackButton />}
  rightButton={<NextButton />}
  variant={'text'}
/>;

<MobileStepper
  activeStep={activeStep}
  steps={steps}
  leftButton={<BackButton />}
  rightButton={<NextButton />}
  variant={'progress'}
/>;
