## Header

The `<Header>` component is used at the top of a page to display information. It displays a title and can optionally show a subtitle, a background image, a navigation button, and multiple action buttons. The header can also be configured to expand or collapse as needed. For expand/collapse behavior tied to the screen scroll position, you should use the [`<CollapsibleHeaderLayout>`](/components/collapsible-header-layout/examples) component.

<Box
  sx={{
    width: '100%',
    textAlign: 'center',
  }}>
  <Box
    sx={{
      width: '100%',
      maxWidth: '500px',
    }}
    component="img"
    alt="Header"
    src={headerImageExample}
  />
</Box>

```
<Header
    title={'Valley Forge'}
    subtitle={'The Last Stand'}
/>;
```

    title={'Valley Forge'}
    subtitle={'The Last Stand'}

    icon={{ name: 'menu' }}
    onIconPress={() => {}}
    actionItems={[
        {
            icon: { name: 'settings'},
            onPress: () => {},
        },
        {
            icon: { name: 'more-vert' },
            onPress: () => {},
        },
    ]}
    variant="static"
    icon={{ name: 'menu' }}
    searchableConfig={{ onChangeText: () => {} }}
    expandable={true}
    collapsedHeight={56}
    expandedHeight={200}
/>;
