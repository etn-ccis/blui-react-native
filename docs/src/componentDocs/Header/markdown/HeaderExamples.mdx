## Header

A basic `<Header>` component is used at the top of the screen and can show a title and (optional) subtitle. A third line of text can also be configured using the `info` prop, but it will only be visible when the `Header` is expanded (see below).

<Header />

## Changing the Background

You can apply a simple background to the Header using the `backgroundColor` prop. You can also apply an image background using `backgroundImage`. If both are applied, they will be blended together.

<HeaderWithBackground />

## Adding Actions

You can add custom icon buttons to the top right corner of the Header. You can also configure an icon in the top left for typical navigation functions (such as a menu icon or a back icon, depending on the context).

<HeaderWithActions />

## Adding a Search Bar

The `Header` includes support for a built-in search bar. If configured, the search button will be appended to any provided action items.

<HeaderWithSearch />

## Expanding and Collapsing

The `Header` can be configured to expand and collapse from a large size to a small size. If you want the `Header` to expand and collapse automatically as the page is scrolled, check out the [`CollapsibleHeaderLayout`](/components/collapsible-header-layout/examples) component.

<HeaderWithExpandCollapse />
