## Header

A basic `Header` component is used at the top of the screen and can show a title and (optional) subtitle. A third line of text can also be configured using the `info` prop, but it will only be visible when the `Header` is expanded (see below). 

<Box
  sx={{
    width: '100%',
    textAlign: 'center',
  }}>
  <Box
    sx={{
      width: '100%',
      maxWidth: '500px',
    }}
    component="img"
    alt="Header"
    src={headerImageExample}
  />
</Box>

```
<Header
    title={'Valley Forge'}
    subtitle={'The Last Stand'}
/>;
```

## Adding Actions

You can add custom icon buttons to the top right corner of the Header. You can also configure an icon in the top left for typical navigation functions (such as a menu icon or a back icon, depending on the context).

<Box
  sx={{
    width: '100%',
    textAlign: 'center',
  }}>
  <Box
    sx={{
      width: '100%',
      maxWidth: '500px',
    }}
    component="img"
    alt="Header actions"
    src={headerActionsExample}
  />
</Box>

```
<Header
    title={'Valley Forge'}
    subtitle={'The Last Stand'}
    actionItems={[
        {
            icon: { name: 'settings'},
            onPress: () => {},
        },
        {
            icon: { name: 'more-vert' },
            onPress: () => {},
        },
    ]}
    icon={{ name: 'menu' }}
/>;
```

## Header with Collapsing and Expanding

<Box
  sx={{
    display: 'flex',
    justifyContent: 'space-evenly',
    alignItems: 'center',
    mb: 2,
    width: '100%',
    textAlign: 'center',
  }}>
  <Box
    sx={{
      width: '40%',
    }}
    component="img"
    alt="Header collapsed"
    src={headerCollapsedExample}
  />
  <Box
    sx={{
      width: '40%',
    }}
    component="img"
    alt="Header expanded"
    src={headerExpandedExample}
  />
</Box>

```
<Header
    title={'Valley Forge'}
    subtitle={'The Last Stand'}
    info={'Information'}
    backgroundImage={farm.jpg}
    icon={{ name: 'menu' }}
    onIconPress={() => {}}
    actionItems={[
        {
            icon: { name: 'settings'},
            onPress: () => {},
        },
        {
            icon: { name: 'more-vert' },
            onPress: () => {},
        },
    ]}
    variant="static"
    icon={{ name: 'menu' }}
    searchableConfig={{ onChangeText: () => {} }}
    expandable={true}
    collapsedHeight={56}
    expandedHeight={200}
/>;
```
