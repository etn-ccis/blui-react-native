# Overline

The Overline component is a specialized [Text](https://callstack.github.io/react-native-paper/docs/components/Text/) component speciaifcally for BLUI applications. It is used in places like Drawer section labels, etc.

## API

<Box>

| Prop Name | Description           | Type                          | Required | Default |
| --------- | --------------------- | ----------------------------- | -------- | ------- |
| theme     | Theme value overrides | `$DeepPartial<ExtendedTheme>` | no       |         |

</Box>

Any other props will be provided to the root element [`<Text>`](https://callstack.github.io/react-native-paper/docs/components/Text).

### Styles

You can override the internal styles used by Brightlayer UI by passing a `styles` prop. It supports the following keys:

| Name | Description                        |
| ---- | ---------------------------------- |
| root | Styles applied to the root element |
